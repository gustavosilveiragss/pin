como enviar pro pin▾
  1. monte e baixe o .pin acima
  2. clique 2x no botão do pin (boot) → abre o menu wifi (mostra o código do pin e o status do encontro)
  3. conecte na rede do pin (o nome do wifi termina com o código do pin, ex broche-A2B3) → a página abre sozinha (senão 192.168.4.1)
  4. envie o .pin
